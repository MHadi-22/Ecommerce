# ⚓ Acuire — Elegant Men's Fashion Store

Editorial warm light-theme e-commerce built with React 18.
Inspired by the Acuire design concept.

## 🚀 Setup

```bash
npm install
npm start
```

Open http://localhost:3000

## ✅ Features

- Warm cream/parchment editorial theme
- Split hero layout (text + fashion photo)
- 8 real product images from Pexels
- Category filter tabs
- Product cards with hover Add-to-Cart
- Slide-in cart drawer
- Quantity controls + remove items
- Complimentary shipping tracker ($150+)
- Lookbook editorial section
- Customer testimonial section
- Newsletter subscription section
- Staggered fade-up animations
- Shimmer loading skeleton
- Fully responsive

## 🎨 Design

Fonts  : Libre Baskerville (headings) + Outfit (body)
Colors : Cream #f5f0e8 · Teal #2d5a4e · Ink #2c2416
